Работа с Bootstrap 4

Март 1, 2019

Bootstrap (Twitter Bootstrap) - это html, css фреймворк* .

Bootstrap 3.3 - официальный сайт более старой версии фреймворка.

Bootstrap 3.3 RU - русскоязычная версия Bootstrap 3.3.

Bootstrap 4.3 - современная версия.

# Введение:

Bootstrap - самый популярный html, css и js фоеймворк*.

Зайдя на оффициальный сайт фреймворка видим описание:

Bootstrap

Build responsive, mobile-first projects on the web with the world’s most popular front-end component library.

Такая популярность Bootstrap возникала благодаря сетке, которая позволяет легко адаптировать любой сайт для мобильных устройств.

Сетка сделана из группирования строк и колонок внутри одного или нескольких контейнеров.

Различают более старую версию Bootstrap 3 и современную версию Bootstrap 4.

Bootstrap 3 сетка основана на float'ах. Т.е блоки располагаются друг за другом благодаря свойству float: left;. Соответственно поддерживается более старыми версиями браузеров и устройств (IE8-9).

Bootstarp 4 сетка основана на flexbox*. Поддержка более современными браузерами и устройствами (IE10-11). Supported browsers Bootstrap 4.3.

Плюсы Bootstrap: кроссбраузерность, адаптивность, единый код для работы в команде, понятная и подробная документация, простая реализация html-шаблона человеком не знакомым с css (задав определенные классы в html можно выровнять элемент, задать фон, цвет и прочее).

Минусы Bootstrap: не все css свойства используются в проекте. Если Bootstrap необходим в проекте для задания сетки, то ее можно скачать и задать отдельно.

# Сетка Bootstrap 4:

Сетки Bootstrap могут использоваться отдельно, без Bootstrap JavaScript и других CSS компонентов. Вам надо только скачать и сослаться на "bootstrap-grid.css", который включает в себя flexbox классы и классы для сетки.

Скачать весь архив с оффициального сайта → Распаковать → Подключить в html необходимый файл.

Документация по сетке Bootstrap 4.

Пример 1 колонка: 

<div class="container">
    <div class="row">
        <div class="col">Первая колонка</div>
    </div>
</div>

Контейнер → строка → колонка

Важно:

  1. Строки row должны быть прямыми потомками контейнера container или container-fluid или родитель с внутренними отступами не менее чем в padding: 15px. Иначе появится горизонтальная прокрутка.
  2. Колонки col должны быть прямыми потомками строк row.
  3. Строки row используются только как обертка для колонок и не для чего больше.

Блок <div class="custom ... - кастомный блок для наглядности примера. Стили заданы с помощью bootstrap классов.

Класс .row обладает свойством display: flex;. Поэтому если добавить еще одну колонку, все будут распологаться на одной строке, но с возможностью переноса на новую, если колокни не будут вмещаться на одной строке по ширине.

Классы начинающиеся с .col - колонки имеют свойство padding-left: 15px; padding-right: 15px;. Поэтому наглядно видны отступы или gutter'ы которые разделяют блоки.

Для того чтобы убрать эти отстпуы, нужно всего лишь добавить класс .no-gutters для строки .row

Если в более старой версии Bootstrap возникала проблема с выравниванием блока в 5 или 7 колонок, то в Bootstrap 4 это легко сделать. Можно вырванивать блоки как горизонтально, так и вертикально. Для этого достаточно добавить строке определенный класс.

Горизонтальное вырванивание начинается с justify-content- (start, end, center, around, between).

Вертикальное вырванивание начинается с align-items- (start, end, center).

Если нужно выровнять определенную колонку, а не все сразу. Добавляется необходимый класс не строке, а именно колонке и начинается с align-self- (start, end, center).
При этом важно чтобы у строки .row была задана высота минимальная или фиксированная.

# Адаптивность сетки Bootstrap 4:

Адаптивные точки Bootstrap:

  • xs — ширина экрана < 576px
  • sm — ширина экрана ≥ 576px
  • md — ширина экрана ≥ 768px
  • lg — ширина экрана ≥ 992px
  • xl — ширина экрана ≥ 1200px

Задание адаптивность колонкам и строкам:

  • .col - колонка на всю доступную ширину
  • .col-auto - колонка с шириной width: auto;
  • .col- - стандарт для всех разрешений
  • .col-sm-
  • .col-md-
  • .col-lg-
  • .col-xl-

# Вложенность

В Bootstrap есть возможность размещать "колонки внутри колонок".
Но при этом не забываем правило:

Колонки col должны быть прямыми потомками строк row.

Т.е. .container → .row → .col- → .row → .col- → ...
